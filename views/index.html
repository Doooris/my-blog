  • git常用命令

    2016-11-28byDoris

    git 是一个强大的版本管理工具，因为它的出现，世界上增添了多少无数充满激情的项目，下面是它一些基本的用法。

    ubuntu下安装初始化
    1. sudo apt-get install git
    2. mkdir demo（新建项目）
    3. cd demo
    4. git init (初始化 git 仓库)

    一个空的版本库就创建好了，当前目录有一个叫 .git 的子目录。

    接下来向版本库中添加文件
    1. touch README
    2. git status （用来查看仓库状态）
    3. git add . (将当前目录添加到 git 仓库中)
    4. git commit -m "Initial commit"(向版本库提交变化)
